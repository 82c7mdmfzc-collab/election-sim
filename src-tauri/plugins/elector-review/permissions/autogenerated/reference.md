## Default Permission

Default permissions for the Elector in-app review bridge

#### This default permission set includes the following:

- `allow-request-review`

## Permission Table

<table>
<tr>
<th>Identifier</th>
<th>Description</th>
</tr>


<tr>
<td>

`elector-review:allow-request-review`

</td>
<td>

Enables the request_review command without any pre-configured scope.

</td>
</tr>

<tr>
<td>

`elector-review:deny-request-review`

</td>
<td>

Denies the request_review command without any pre-configured scope.

</td>
</tr>
</table>
