## Default Permission

Default permissions for the Elector Sign in with Apple bridge

#### This default permission set includes the following:

- `allow-sign-in-with-apple`

## Permission Table

<table>
<tr>
<th>Identifier</th>
<th>Description</th>
</tr>


<tr>
<td>

`elector-siwa:allow-sign-in-with-apple`

</td>
<td>

Enables the sign_in_with_apple command without any pre-configured scope.

</td>
</tr>

<tr>
<td>

`elector-siwa:deny-sign-in-with-apple`

</td>
<td>

Denies the sign_in_with_apple command without any pre-configured scope.

</td>
</tr>
</table>
