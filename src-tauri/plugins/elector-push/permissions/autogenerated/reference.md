## Default Permission

Default permissions for the Elector push registration bridge

#### This default permission set includes the following:

- `allow-register-for-push`

## Permission Table

<table>
<tr>
<th>Identifier</th>
<th>Description</th>
</tr>


<tr>
<td>

`elector-push:allow-register-for-push`

</td>
<td>

Enables the register_for_push command without any pre-configured scope.

</td>
</tr>

<tr>
<td>

`elector-push:deny-register-for-push`

</td>
<td>

Denies the register_for_push command without any pre-configured scope.

</td>
</tr>
</table>
